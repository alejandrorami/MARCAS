TITULO

<!doctype html>

<html lang=“es”>

<head>

<title>Mi primer documento HTML</title>

</head> <body> Este es el texto de mi primer documento. </body>

</html>

Este es el titulo del documento, que parecera en la ventana del navegador

METADATOS

<!doctype html>

<html lang=“es”>

<head>

<title>Mi primer documento HTML</title>

<meta charset=“utf-8”>

<meta name=“author” content=“Antonio”>

</head>

<body> Este es el texto de mi primer documento. </body>

</html>

Estos son los metadatos, y sirve para dar informacion del documente, como la codificacion o el nombre del creador

TITULOS

HEADING: H

<h1>Texto de título de primer nivel</h1>

<h2>Texto de título de segundo nivel</h2>

<h6>Texto de título de sexto nivel</h6>

Puede haber indefinidamente, y sirve para crear un titulo, se pone automaticamente en un tamaño mayor y en negrita

    AGRUPACION

  • Párrafo: P

    • <p>Este texto pertenece a un párrafo.</p>

    <p>Pero éste es del párrafo siguiente</p>

  • Texto preformateado: PRE

    • <pre>

    Este texto aparecerá

    escrito en el navegador con este mismo formato

    utilizando un tipo de letra monoespaciada.

    </pre>

    Esto hace que aparezca unos saltos de linea al inicio y al final del texto

  • Listas: LI

    • Listas desordenadas: ul

    Listas ordenadas: ol

    Definition list: dl

    Este elemento sirve para crear una lista, ya sea desordenada o ordenada

  • Linea horizontal: HR

    • <p>La siguiente línea horizontal divide temáticamente este párrafo del siguiente</p>

    <hr>

    <p>Este segundo párrafo puede pertenecer, por ejemplo, a un cambio de escena dentro de un mismo capítulo de una novela.</p>

    Este elemento sirve para espaciar dos parrafos con una linea horizontal, se podria usar para novelas como bien dice el ejemplo

    Elementos semánticos de texto

  • Resaltado:

    • Italic: I- Bastardilla

    <p>Este párrafo contiene una palabra en cursiva en su interior: la escribimos con el símbolo de <i>itálica</i>.</p>

    Bold: B- Negrita

    <p>En este, sin embargo, hay una palabra que aparece en <b>negrita</b>.</p>

    Underlined: U- Subrayada

    <p>Y la palabra <u>subrayada</u> es la que podemos ver en este párrafo final.</p>

    STRESS EMPHASIS: EM --;Pone énfasis o recalca una parte del texto

    STRONG--Parte importante, seria o urgente

    SMALL--Representa la “letra pequeña” (de un contrato...)

    S--Representa contenido irrelevante u obsoleto

  • Salto de linea

    • BREAK: BR

    WORD BREAK:WBR

    <p>Este párrafo continene un <br>

    salto de línea</p>

  • Hiperenlace:

    • Anchor: A

    Enlace a URL: href

    Esto indica el lugar en el que hay un enlace

    Este texto contiene un enlace a la página de <a href=“http://www.google.com”>Google</a> , la empresa de la gran G.

    Absoluto: http://sitio/pagina.html

    Relativo:

    --Mismo directorio-pagina.html

    --Subdirectorio-Subdirectorio/pagina.html

    --Directorio anterior-../pagina.html

  • Hiperenlace interno:

    • --Identificador de elemento

    <elto id=eltoX>Elemento X</elto>

    --Referencia a elemento

    <a href=#eltoX>Elemento X</a>

    Principio del documento

    #top

  • Hiperenlace general:

    • Referencia desde otro documento

    Pero, por supuesto, también podría acceder a un <a href=“URL#identificador”>lugar</a> concreto de otro documento diferente.

    Debo indicar cuál es la URL del documento y concatenar a la misma el identificador de ese lugar.

  • Atributos del hiperenlace(usando href)

    • -Download

    -Target

    -Otros
